# Install
```bash
mvn clean install
```

## Install senza test Unitari
```bash
mvn clean install -DskipTests
```

## Install senza test Integrazione
```bash
mvn clean install -DskipITs
```

## Install senza test Unitari e Integrazione
```bash
mvn clean install -DskipTests -DskipITs
```

## Run Springboot
```bash
mvn springboot:run
```
